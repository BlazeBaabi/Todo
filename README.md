# Todo
a local ran to do list for your projects that's designed to be quick and simple.

### **how to set up**
- Extract the zip folder
- run the main.pyw file

### **collections**
on the top of your screen
- click on "Default" to view collections
- click on "new" to create a collection
- to rename a collection click "rename"

### **tasks**
on the bottom of your screen
- click on "add" to create a new task
- when creating a task click "save" to add it
- to edit a task click "edit" when selecting a task
- to complete a task click on "complete"
